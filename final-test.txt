## Gemini model test

1. 최신 인공지능 뉴스 3개 찾아서 관련 이미지랑 같이 요약한 뉴스카드 만들어줘.
- 기대효과: 인터넷에서 최신 정보를 찾고 정확하게 그 정보를 요약해서 이미지를 fetching 하고 재구성하여 사용자에게 보여줄 수 있는 능력을 확인.

- 현재: 기대한 대로 요약과 관련 이미지가 나왔음. 

2. 이 문서 분석하고 interactive 하게 시각화해서 이해 쉽게 보여줘.
- 기대효과: 문서(혹은 다양한 입력들)을 읽고 어떻게 처리해서 보여줄 수 있는지 확인

- 현재: 적절하게 잘 나옴. 이대로 유지. babel 에러가 나긴 하지만 나오긴 함. media/gemini에 관련된 이미지가 생성되어있음. 서버 로그에 이미지 생성 관련 메세지 없었는데도 자동으로 생성되어있음 확인 필요.

3. 간략한 소설 제작해줘 중간에 이미지 생성해서 넣어서 만들어줘.
- 기대 효과: 사용자에게 적절한 위치에 이미지 생성을 넣어서 글로 전달하기 힘든 부분과 상상력을 도와서 어떤 요약이나 설명에 대해서도 사용자 편의를 가지고 보여줄 수 있는 능력 확인.

- 현재: 새롭게 생성된 이미지를 잘 가져옴. 지금 상태 좋음. 유지. [[0] [Image] Generation failed (gemini): Gemini image generation returned no image data] 이런게 뜸. 이미지를 2개 생성하고 하나는 fallback 함. 

- 통합 의견: 전반적으로 잘 되고있음. 이미지 생성만 마구잡이로 하지만 않으면 좋을듯.

## openai model test

1. 최신 인공지능 뉴스 3개 찾아서 관련 이미지랑 같이 요약한 뉴스카드 만들어줘.
- 기대효과: 인터넷에서 최신 정보를 찾고 정확하게 그 정보를 요약해서 이미지를 fetching 하고 재구성하여 사용자에게 보여줄 수 있는 능력을 확인.

- 현재: 일단 결과적으로 뉴스를 잘 가져오고 요약도 함. 그런데 3개 했는데 2개는 이미지가 기사에 있음에도 불구하고 fallback 됨. 이건 웹페이지 fetching이 막힌것 안막힌것 차이로 보임. verge에서는 이미지 가져와서 보임.

2. 이 문서 분석하고 interactive 하게 시각화해서 이해 쉽게 보여줘.
- 기대효과: 문서(혹은 다양한 입력들)을 읽고 어떻게 처리해서 보여줄 수 있는지 확인

- 현재: 뭔가 파싱 실패해서 에러난것처럼 보임. type 무슨 에러나고 빨간 화면 막 나옴. 에러 로그 복사 실패. 그리고 media/openai에 보면 생성 이미지가 보임. gemini 처럼 이미지 생성 메세지 없었는데도 자동 생성 되어있음. 전체적으로 gemini랑 openai 모두 이미지 생성 관련 강하게 해야할듯.

3. 간략한 소설 제작해줘 중간에 이미지 생성해서 넣어서 만들어줘.
- 기대 효과: 사용자에게 적절한 위치에 이미지 생성을 넣어서 글로 전달하기 힘든 부분과 상상력을 도와서 어떤 요약이나 설명에 대해서도 사용자 편의를 가지고 보여줄 수 있는 능력 확인.

- 현재: 완벽함.

- 통합의견: 전반적으로 좋은편임. 2번 프롬프트의 타입 무슨 문제가 뭐엿는지 정확히 모르겠음 계속 생성되는거에 type 무슨 에러에 genui 창이 빨간색 뭐가 나온거 봐서는 json 파싱이 잘못된거 같음.
